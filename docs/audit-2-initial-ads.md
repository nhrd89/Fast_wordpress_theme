# Audit 2: Layer 1 — Initial Viewport Ads (`initial-ads.js`)

## Boot Sequence

```
document.readyState === 'complete' → setTimeout(boot, 100)
  └─ boot()
      ├─ Guard: plAds exists? initial-ad-1 container exists?
      └─ loadGPT()
          ├─ If googletag.apiReady: reuse, skip <script>
          └─ Create <script async> for securepubads.g.doubleclick.net/tag/js/gpt.js
              └─ onload → initSlots()
                  ├─ Destroy any stale cached slots
                  ├─ enableSingleRequest() + collapseEmptyDivs()
                  ├─ Define ALL slots (overlays → in-content → sidebar → pause)
                  ├─ enableServices()
                  ├─ googletag.display() for all display slots
                  ├─ Mark window.__initialAds.ready = true
                  ├─ Fire all onReady callbacks
                  ├─ Start event flush (10s interval + pagehide + visibilitychange)
                  └─ Register overlay setInterval refresh timers
```

## `defineSlot` vs `defineOutOfPageSlot`

| Method | Used For | Div ID | Size | Display Method |
|--------|----------|--------|------|---------------|
| `defineSlot(path, sizes, divId)` | In-content slots, sidebar, pause | Explicit (`initial-ad-1`, etc.) | Array of [w,h] pairs | `googletag.display(divId)` — requires DOM element |
| `defineOutOfPageSlot(path, format)` | Overlays (anchor, interstitial, side rails) | Auto-generated by GPT | N/A (format determines) | GPT auto-renders at its discretion |

Key difference: `defineOutOfPageSlot` returns `null` if the format is already defined or unsupported on the current page. The code guards against this with `if (interstitial) { ... }`.

## Refresh Logic

### Display Slots (initial, sidebar, pause)
Triggered by `impressionViewable` GPT event:

```
impressionViewable fires
  └─ Check maxRefresh: 0=never, -1=unlimited, N=cap
  └─ Check refreshCount < maxRefresh
  └─ Get delay from refDelay(slotType) via optimizer config
  └─ setTimeout(delay)
      └─ Guard: tab visible? (VW_CFG.skipTabHidden)
      └─ Guard: viewport check? (VW_CFG.viewportCheck)
          └─ 200px margin above/below, min 10% visible
      └─ googletag.pubads().refresh([slot])
      └─ info.refreshCount++
      └─ trackAdEvent('refresh', divId)
```

### Overlay Slots (anchor, side rails)
Triggered by `setInterval` (NOT `impressionViewable`):

```
setInterval(oDelay) registered after all slots defined
  └─ Guard: document.hidden? Return.
  └─ Guard: maxRefresh reached? Return.
  └─ Guard: overlay status must be 'filled' or 'viewable'
  └─ googletag.pubads().refresh([slotRef])
  └─ oInfo.refreshCount++
  └─ trackAdEvent('refresh', divId)
```

### Interstitial
- `maxRefresh: 0` — NEVER refreshed. GPT interstitials are one-shot.
- No `setInterval` registered for interstitial.
- Exit-intent interstitial (in smart-ads.js) handles re-triggering by destroying the slot and creating a new one.

### Refresh Defaults per Slot Type

| Slot Type | Default Max Refresh | Default Delay | Source |
|-----------|-------------------|---------------|--------|
| initial | 3 | 45s | `refMax('initial', 3)`, `refDelay('initial')` |
| sidebar | 3 | 45s | `refMax('sidebar', 3)`, `refDelay('sidebar')` |
| pause | 2 | 45s | `refMax('pause', 2)`, `refDelay('pause')` |
| anchor | unlimited (-1) | 30s | `refMax('anchor', -1)`, `refDelay('anchor')` |
| sideRail | unlimited (-1) | 30s | `refMax('sideRails', -1)`, `refDelay('sideRail')` |
| interstitial | 0 (never) | N/A | Hardcoded `maxRefresh: 0` |

All configurable via `plAds.refresh` from `inc/ad-optimizer.php`.

## Slot Map Structure (`_slotMap`)

Each entry keyed by divId:
```js
{
    slot:          googletag.Slot,  // GPT slot reference
    type:          'initial' | 'sidebar' | 'pause' | 'interstitial' | 'anchor' | 'sideRail',
    refreshCount:  0,               // current refresh count
    lastRefresh:   0,               // timestamp of last refresh
    maxRefresh:    3,               // -1=unlimited, 0=never, N=cap
    renderedSize:  [300, 250],      // set on slotRenderEnded (null if empty)
    viewable:      false            // set to true on impressionViewable
}
```

## GPT Event Handlers

### `slotRenderEnded` → `onSlotRenderEnded(event)`

1. Updates `__plOverlayStatus` for overlay slots (by both divId and slot reference)
2. If `event.isEmpty`:
   - Collapse container: `minHeight:0, margin:0, overflow:hidden`
   - Track: `pushEvent('ad_empty')` + `trackAdEvent('empty', divId)`
3. If filled:
   - Resize container: only GROW min-height (never shrink — prevents CLS)
   - Set `maxWidth` to creative width
   - Store `renderedSize` in `_slotMap`
   - Track: `pushEvent('ad_filled')` + `trackAdEvent('impression', divId)`

### `impressionViewable` → `onImpressionViewable(event)`

1. Updates `__plOverlayStatus` to `'viewable'` for overlay slots
2. Finds slot info in `_slotMap` (by divId or slot reference fallback)
3. If not found → returns (dynamic slots handled by smart-ads.js)
4. Sets `info.viewable = true`
5. Increments `window.__plViewableCount` (shared with Layer 2)
6. Track: `pushEvent('ad_viewable')` + `trackAdEvent('viewable', divId)`
7. For overlays: returns (refresh handled by setInterval)
8. For display slots: schedules refresh via `setTimeout` with viewport/tab guards

## Event Tracking (`__plAdTracker`)

Exposed as `window.__plAdTracker` with:
- `.track(eventType, slotId, extra)` — queues event to `_eventBatch`
- `.flush()` — sends batch via `navigator.sendBeacon` to `plAds.ajaxUrl + '?action=pl_ad_event'`
- `.sessionId` — random session ID (`pl_` + 12 random chars)
- `.device` — `desktop` / `tablet` / `mobile`
- `.pageType` — `single` / `home` / `archive`

Batch envelope includes: `sid`, `device`, `pageType`, `postId`, `scrollPct`, `timeOnPage`, `visitorType`, `events[]`.

Flush triggers: every 10s interval, `visibilitychange` (hidden), `pagehide`.

### Event Types Tracked by Layer 1

| Event Type | When | Extras |
|-----------|------|--------|
| `impression` | `slotRenderEnded` (filled) | `creativeSize` |
| `empty` | `slotRenderEnded` (isEmpty) | — |
| `viewable` | `impressionViewable` | — |
| `refresh` | After successful `googletag.pubads().refresh()` | — |
| `refresh_skip` | Refresh aborted (tab hidden or out of viewport) | — |

Layer 2 (smart-ads.js) uses the same `__plAdTracker.track()` API with additional fields: `injectionType`, `scrollSpeed`, `scrollDirection`, `adSpacing`, `nearImage`, `adsInViewport`, `adDensityPercent`, `gptResponseMs`, `relocated`.

## Overlay Status State Machine

```
'off' (format disabled in admin)
  ↓ (format enabled)
'pending' (defineOutOfPageSlot called, waiting for GPT)
  ↓ slotRenderEnded
'empty' (isEmpty: true)  OR  'filled' (isEmpty: false)
  ↓ impressionViewable
'viewable'
```

Overlay refresh (setInterval) only fires when status is `'filled'` or `'viewable'`.
Exit-intent interstitial checks status before acting:
- `'pending'` → refresh existing slot to nudge GPT
- `'filled'`/`'viewable'`/`'empty'` → destroy and create new exit slot
